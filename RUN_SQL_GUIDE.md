# HOW TO RUN SQL — step by step

## Step 1: Open Supabase
1. Go to https://supabase.com/dashboard
2. Sign in
3. Click on your project (yun store)

## Step 2: Open SQL Editor
1. On the left sidebar, click **SQL Editor**
2. Click **New query** (top right)

## Step 3: Run shipping-prices.sql
1. Open the file `shipping-prices.sql` on your computer
2. Copy ALL the content
3. Paste it in the SQL Editor
4. Click **Run** (or press Ctrl+Enter)
5. You should see "Success" at the bottom

## Step 4: Run fix-admin.sql
1. Click **New query** again
2. Open `fix-admin.sql` on your computer
3. Copy ALL the content
4. Paste it in the SQL Editor
5. Click **Run**

## Step 5: Run add-checkout-fields.sql
1. Click **New query** again
2. Open `add-checkout-fields.sql` on your computer
3. Copy ALL the content
4. Paste it in the SQL Editor
5. Click **Run**

## Done!
All 3 SQL files are now running on your database.
